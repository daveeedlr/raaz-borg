# RAAZ BORG — Combat Re-Coded

Official website of **Ra'az Undergrounds** — an underground grappling platform from Guayaquil, EC, where traditional martial arts collide with neuro-interface fiction.

## Pages

| Page | File |
|---|---|
| Universe (home) | `index.html` |
| Fighters | `fighters.html` |
| Clashes GP | `clashes.html` |
| Labs & Media | `labs.html` |
| Shop — Combat Suits | `shop.html` |

## Stack

Static HTML/CSS/JS — no build step. Shared styles in `css/style.css`, shared logic (EN/ES language toggle, glitch/ticker/reveal effects, cart counter) in `js/main.js`, optimized images in `assets/`.

## Local preview

Open `index.html` in a browser, or:

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

## Deploy

Works out of the box on GitHub Pages (Settings → Pages → Deploy from branch → `main`, root).
